BMW PRICE PREDICTION USING MACHINE LEARNING

1. Project Overview
This project predicts the selling price of BMW cars based on features such as model, year, mileage, fuel type, transmission, and engine specifications. The objective is to build a regression model that can estimate the market price of a BMW car accurately.

2. Dataset
Source: https://www.kaggle.com/datasets/nalisha/bmw-car-sales-and-price-dataset
Number of rows: 10781
Number of columns: 9

Features:
- Model
- Year
- Tax
- mpg
- Mileage
- Fuel Type
- Transmission
- Engine Size

Target Variable:
- Price

3. Data Preprocessing
- Removed missing values using SimpleImputer.
- Encoded categorical features using OneHotEncoder.
- Used ColumnTransformer to preprocess numerical and categorical columns.
- Built a preprocessing pipeline using Scikit-learn.

4. Models Trained
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Neural Network
